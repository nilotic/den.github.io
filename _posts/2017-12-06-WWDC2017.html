Graphics and Game

🔍 Introducing Metal 2

  • Argument Buffer 
    (기존 Argument Buffer 처리하는 단계가 많은데 간소화 시켜서 Performance 향상. 실시간 Material, Particle 변경하는데 좋음, Demo 참고)
  • Raster Order Groups 
    (Shader blend 할 때, Thread 간에 Bubble이 생기는데 group으로 묶어서 Performance 향상.)
  • ProMotion Displays 
    (GPU와 Display 간의 처리 시간 차이로 frame 저하 형상이 생김. 이걸 여러 가지 방법으로 최적화 시킴) 👍👍👍
  • Direct to Display 
    (Metal View → System Compositor → Display 과정을. Metal View → Display 로 바로 접근 가능하도록 함. Instrument에서 동작 확인 가능)
  • Everything Else 
    (MTLResource, MTLHeap, MTLDevice Memory size 조정 가능, SIMDGroup-Scoped Data Sharing, Threadgroup Size 조절 통해 Memory 최적화, ViewPort Arrays, MultiSample Pattern Control, Resource Heaps)

⭐️ Introducing ARKit: Augmented Reality for iOS 👍

  • ARKit 
    (A9 칩부터 지원. [Tracking, Scene Understanding, Rendering] 3가지로 나누어짐. )
  • ARSession 
    (URLSession 과 비슷한 개념, Configuration 바꾸고 Run 실행하면 끊어짐 없이 바로 적용)
  • ARFrame (Session 에서 생성된 각 Frame 들)
  • ARAnchor (AR world 에 있는 anchor)
  • ARWorldTrackingSessionConfigureation 
    (URLSession Configuration 과 비슷한 개념. Interface 가 URLSession 과 비슷해서 편한듯.)
  • ARCamera (3D World의 camera와 동일한 개념)
  • Tracking State를 통해 상태를 알 수 있음 (Delegate 참고)
  • Plane Detection 
    (Detection 과정은 영상 참고. configuration.planeDetection = .horizontal 통해서 설정 가능. )
  • ARPlaneAnchor 
    (Delegate 확인, Called when a plane’s transform or extent is updated. Sample Code 필수 참고)
  • Light Estimation (주변 빛의 따라 가상 Object Light 속성 변경 가능.)
  • SceneKit 에서 AR 실행되는 과정 설명 
    (SCNView → ARSCNView → ARSession)
  • SpriteKit 에서 AR 실행되는 과정 설명 (Maps SKNodes → ARAnchors)
  • Custom Rendering 
    (Custom renderer 개발자를 위한 Tips, Draw camera background, Update virtual camera, Update lighting, Update transforms for geometry)

🔍 SceneKit: What’s New

  • Camera Enhancements 
    (HDR camera 가능, Object Motion Blur 추가 ), Demo 참고 👍
  • Ambient Occlusion (Screen space ambient occlusion)
  • Camera Control 
    [ Debug 때 좋음. Camera로 날아다니며 여기 저기 확인 가능 ], 
    여러 옵션에 따라 Camera 설정 가능 [ 따라다니기, 고정, 따라다니기 + 고정, etc ], Demo 참고 👍
  • Tessellation (Tessellation 옵션 설정해서 적용 가능.), Demo 참고 👍
  • Displacement Mapping, 영상 참고
  • Subdivision Surfaces 
    (easier and faster to create by artists, Reduced file sizes and faster load times, Dynamic control on quality of rendered geometry , Level 조절만으로 Object 가 엄청 부드러워짐) 자료 참고
  • Animation Blending (Step, Walk, Run, Animation을 blend)
  • Developer Tools 
    (SceneKit Instrument 추가, View Capture button 누르면, View 나오고 거기서 다시 SCNScene 누르면 Current Scene 3D로 보여줌)
  • SceneKit Scene Editer 
    (SCNNodeReference에 추가 Override Material 적용 가능)
  • UIFocus Support (Focus Engine Focus 처리를 알아서 해준다)
  • Point Cloud Rendering 추가.
  • Transparency Modes (.default, .singleLayer, .dualLayer) 추가
  • Cascaded Shadow Maps (값 조절에 따라 일정 영역만 적용 가능) 자료 참고
