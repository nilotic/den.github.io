Tech Talks

HDR10

  • Static metadata (SEI NAL units) should be in the HEVC Configuration Box

Dolby Vision

  • Profile 5
    - Single layer
    - 10-bit HEVC
  • Up to level 7

HDR

  • HEVC
  • Include variants ≤ 30FPS
  • SDR variants must be provided
  • SDR I-frame variants must be provided

New Tag VIDEO-RANGE

  • Must be included if playlist includes HDR content 
    - SDR: Standard dynamic range variants
    - PQ: HDR10 or Dolby Vision variants
  • HDCP-LEVEL
    Specifies the minimum HDCP level required for secure playback
    - NONE
    - TYPE-0
    - TYPE-1
